Detailed Description

template<typename internal_int_t, std::size_t Q>
class fixedpointnumber::fixed_t< internal_int_t, Q >

Type for fixed point number.

This template class provides fixed point number with specified integer type and specified Q value.

Template Parameters
internal_int_tInternal integral type to hold fixed point number
QBits width for decimal part
